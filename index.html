How to play the game

A ramdom number will be given at start the game.

There are four crystals.When a crystal is clicked a specific amount of points will be added to your total score.

To win the game player has to match the total score to random number.If player goes above the random number then player will lose the game.

The value of a crystal will be hidden until it is clicked.

Each time when the game starts, the game will change the values of each crystal.
